# Упражнения на ввод данных и условные операторы

Это серия заданий для отработки навыков работы с вводом данных от пользователя (`input()`) и использования условных конструкций (`if`, `elif`, `else`).

## Цель

Вам предстоит решить несколько задач в файле `main.py`, каждая из которых посвящена определенной теме. После этого вы запустите тесты для проверки своих решений.

## Инструкция

### Шаг 1: Настройте окружение

```bash
python -m venv venv
source venv/Scripts/activate
pip install -r requirements.txt
```

### Шаг 2: Выполните задания в `main.py`

Откройте `main.py` и реализуйте логику для следующих функций:

1.  **`ask_for_age()`**: Запрашивает у пользователя возраст и, в зависимости от значения, выводит одно из сообщений: "Вы совершеннолетний" или "Вы несовершеннолетний".

2.  **`check_password()`**: Запрашивает пароль и проверяет, совпадает ли он с "secret". Выводит "Пароль верный" или "Пароль неверный".

3.  **`check_temperature()`**: Запрашивает температуру и, в зависимости от диапазона, выводит "Холодно" (ниже 15), "Тепло" (от 15 до 25) или "Жарко" (выше 25).

4.  **`check_login_and_password()`**: Запрашивает логин и пароль. Доступ разрешен только если логин — "admin" и пароль — "password123".

### Шаг 3: Запустите тесты

Для проверки каждой задачи написаны тесты. Запустите их командой:

```bash
pytest
```

Тесты будут вводить данные в вашу программу и проверять, соответствует ли вывод ожидаемому результату. Если что-то пойдет не так, `pytest` предоставит детальный отчет.
